<objective>
Add MVCC multi-version storage to the KV store. The current implementation stores only the latest value per key (single-version storage: `HashMap<Vec<u8>, KvEntry>`), which prevents true snapshot isolation where older snapshots should see older versions of keys.

**Purpose:** Enable true snapshot isolation where each snapshot sees the data as it existed at that point in time, not just "latest or nothing".

**Problem:** Current implementation overwrites old versions. If key="foo" has version 100, then version 200, then version 300:
- Snapshot at LSN 250 should see version 200
- Currently sees nothing (because version 300 > 250, and version 200 was overwritten)

**Solution:** Change storage to retain version history per key:
- `HashMap<Vec<u8>, KvEntry>` -> `HashMap<Vec<u8>, Vec<KvEntry>>`
- Versions stored in LSN order (ascending)
- `get_at_snapshot()` uses binary search to find latest version <= snapshot_lsn
- Old versions retained (no automatic cleanup)
- TTL checked on the visible version only

**Output:** True MVCC snapshot isolation with O(log n) version lookup, backward compatible with existing tests.
</objective>

    Binary search approach:
    - Vec<KvEntry> is sorted by version (ascending)
    - Use partition_point or manual binary search
    - Find rightmost entry where entry.version <= snapshot_lsn
    - Return that entry (or None if no such entry)

    Pseudocode:
    ```rust
    pub fn get_at_snapshot(&self, key: &[u8], snapshot_id: SnapshotId) -> Result<Option<KvValue>, KvStoreError> {
        let entries = self.entries.read();
        let snapshot_lsn = snapshot_id.as_lsn();

        if let Some(versions) = entries.get(key) {
            // Binary search for latest version <= snapshot_lsn
            let idx = versions.partition_point(|e| e.metadata.version <= snapshot_lsn);
            if idx == 0 {
                return Ok(None);  // All versions too new
            }
            let entry = &versions[idx - 1];

            // Check TTL
            if ttl::is_expired(entry) {
                return Ok(None);
            }
            Ok(Some(entry.value.clone()))
        } else {
            Ok(None)
        }
    }
    ```

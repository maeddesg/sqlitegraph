# Key decisions from research
- Natural loop: back-edge (tail -> header) where header dominates tail
- Loop body: all nodes reachable from tail without passing through header (DFS from tail, skip header)
- Multiple back-edges to same header: group into single loop with multiple back_edges entries
- Nested loops: inner loop header is in outer loop's body
- Irreducible CFG: edges where neither end dominates the other are NOT natural loops
- Nesting tree: parent headers -> vector of child headers (direct children only)
- Loop body excludes header (header is separate field)

# DominatorResult API from Phase 47
- DominatorResult.dominates(dominator, node): bool - check if dominator dominates node
- DominatorResult.idom: AHashMap<i64, Option<i64>> - immediate dominator tree

```rust
use ahash::AHashMap;
use ahash::AHashSet;

/// A natural loop identified by back-edges
#[derive(Debug, Clone)]
pub struct NaturalLoop {
    /// Loop header (dominates all loop nodes)
    pub header: i64,
    /// All back-edges to this header: (tail -> header)
    pub back_edges: Vec<(i64, i64)>,
    /// Loop body nodes (excluding header)
    pub body: AHashSet<i64>,
}

impl NaturalLoop {
    /// Check if a node is in this loop (header or body)
    pub fn contains(&self, node: i64) -> bool {
        self.header == node || self.body.contains(&node)
    }

    /// Get all nodes in the loop (header + body)
    pub fn all_nodes(&self) -> AHashSet<i64> {
        let mut nodes = self.body.clone();
        nodes.insert(self.header);
        nodes
    }

    /// Check if this loop is nested within another
    pub fn is_nested_in(&self, parent: &NaturalLoop) -> bool {
        parent.contains(self.header)
    }

    /// Get nesting depth of this loop within another (0 = not nested)
    pub fn nesting_depth_in(&self, parent: &NaturalLoop) -> usize {
        if !parent.contains(self.header) {
            return 0;
        }
        // Check if there's an intermediate loop
        // For simplicity, return 1 if nested, 0 otherwise
        // Full implementation would check against all loops
        1
    }
}

/// Natural loop detection result
#[derive(Debug, Clone)]
pub struct NaturalLoopsResult {
    /// Map from header to its natural loop
    pub loops: AHashMap<i64, NaturalLoop>,
}

impl NaturalLoopsResult {
    /// Get the loop with this header (if any)
    pub fn loop_with_header(&self, header: i64) -> Option<&NaturalLoop> {
        self.loops.get(&header)
    }

    /// Check if a node is a loop header
    pub fn is_loop_header(&self, node: i64) -> bool {
        self.loops.contains_key(&node)
    }

    /// Get loop nesting depth for a node (0 = not in loop)
    pub fn nesting_depth(&self, node: i64) -> usize {
        self.loops.values().filter(|l| l.contains(node)).count()
    }

    /// Get all loops at a given nesting depth
    pub fn loops_at_depth(&self, depth: usize) -> Vec<&NaturalLoop> {
        self.loops.values()
            .filter(|l| self.nesting_depth(l.header) == depth)
            .collect()
    }

    /// Build loop nesting tree (parent headers -> child headers)
    pub fn nesting_tree(&self) -> AHashMap<i64, Vec<i64>> {
        let mut tree: AHashMap<i64, Vec<i64>> = AHashMap::new();

        for (&header, loop_) in &self.loops {
            for (&potential_parent, parent_loop) in &self.loops {
                if header == potential_parent {
                    continue;
                }
                if parent_loop.contains(header) {
                    // Check if this is the direct parent (no intermediate loop)
                    let is_direct = !self.loops.values()
                        .any(|other| {
                            other.header != potential_parent
                                && other.header != header
                                && parent_loop.contains(other.header)
                                && other.contains(header)
                        });

                    if is_direct {
                        tree.entry(potential_parent)
                            .or_insert_with(Vec::new)
                            .push(header);
                    }
                }
            }
        }

        tree
    }

    /// Get total number of loops detected
    pub fn count(&self) -> usize {
        self.loops.len()
    }

    /// Get all loop headers
    pub fn headers(&self) -> Vec<i64> {
        self.loops.keys().copied().collect()
    }
}
```

3. Core algorithm:
   - `compute_loop_body(graph, header, tail) -> AHashSet<i64>`
   - Algorithm: DFS from tail, add all reachable nodes except header
   - `natural_loops_internal(graph, dom_result, all_nodes) -> NaturalLoopsResult`
   - Algorithm:
       * For each edge (tail, header) in graph:
           - if dom_result.dominates(header, tail): back-edge found
           - if loops[header] doesn't exist: create new NaturalLoop
           - Add (tail, header) to loops[header].back_edges
           - Compute loop body via DFS from tail
   - Entry node check: Skip edges where entry is tail (entry can't be tail of back-edge)

5. Error handling:
   - Handle empty graphs (return empty result)
   - Handle single node (no loops possible)
   - Handle unreachable nodes (skip in edge iteration)

Pattern:
```rust

        // Multi-layer mode: use LayerMappings

            .ok_or_else(|| HnswError::Index(HnswIndexError::NodeNotFound(vector_id)))?
    } else {
        // Single-layer mode: direct conversion
        vector_id - 1
    };

    // ... rest of the method using node_id
}
```

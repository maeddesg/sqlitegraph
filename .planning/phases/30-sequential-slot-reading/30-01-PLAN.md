1. **Method signature** (after read_node_v2(), around line 390):
   ```rust
   /// Read multiple sequential node slots in a single I/O operation
   ///
   /// # Parameters
   /// - `start_node_id`: First node ID to read (must be >= 1)
   /// - `count`: Number of sequential slots to read (max 8 recommended)
   ///
   /// # Returns
   /// Vector of successfully decoded NodeRecordV2 instances
   ///
   /// # Preconditions
   /// - All node IDs must be valid (>= 1 and <= node_count)
   /// - start_node_id + count - 1 <= node_count (clamped internally)
   ///
   /// # I/O Behavior
   /// Single read_exact() call for all slots (e.g., 8 slots = 32KB in one syscall)
   pub fn read_slots_batch(
       &mut self,
       start_node_id: NativeNodeId,
       count: usize,
   ) -> NativeResult<Vec<NodeRecordV2>>
   ```

2. **Validation logic** (return early errors):
   - Check `start_node_id >= 1` else return InvalidNodeId error
   - Check `start_node_id <= header.node_count` else return InvalidNodeId error
   - Clamp `count` to available nodes: `actual_count = count.min(available)`
   - Where `available = (header.node_count as NativeNodeId - start_node_id + 1) as usize`

3. **Bounds checking** (prevent overflow):
   - Calculate `total_bytes = (actual_count as u64).checked_mul(NODE_SLOT_SIZE)`
   - Return CorruptNodeRecord error if overflow (use ok_or())
   - Use `constants::node::NODE_SLOT_SIZE` (4096) explicitly

4. **File size validation**:
   - Calculate `start_offset = node_data_offset + ((start_node_id - 1) as u64 * NODE_SLOT_SIZE)`
   - Get `file_size = self.graph_file.file_size()?`
   - Return FileTooSmall error if `start_offset + total_bytes > file_size`

5. **Single batch read**:
   - Allocate `buffer = vec![0u8; total_bytes as usize]`
   - Call `self.graph_file.read_bytes(start_offset, &mut buffer)?`
   - This is the KEY optimization: 1 syscall instead of 8

6. **Decode each slot** (loop over actual_count):
   - For each slot i: calculate `slot_start = i * NODE_SLOT_SIZE as usize`
   - Extract slice: `&buffer[slot_start..slot_start + NODE_SLOT_SIZE as usize]`
   - Call `NodeRecordV2::deserialize(slot_data)?` for each
   - Push to results Vec

7. **Return** `Ok(results)` with all decoded records

**DO NOT**:
- Use unwrap() or expect() - all errors must be properly handled
- Skip bounds checking - this is critical for file safety
- Use hardcoded 4096 - always use NODE_SLOT_SIZE constant
- Read beyond file bounds - clamp count to available nodes

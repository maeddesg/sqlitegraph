Output: Two new public functions (louvain_communities, label_propagation) with tests.

  - Signature: pub fn louvain_communities(graph: &SqliteGraph, max_iterations: usize) -> Result<Vec<Vec<i64>>, SqliteGraphError>
  - Algorithm (simplified single-pass version):
    1. Initialize: each node in its own community (community_id = node_id)
    2. For each iteration up to max_iterations:
       - For each node in random order:
         - Calculate modularity delta if node moves to each neighbor's community
         - Move node to community with maximum positive delta (if any)
         - Track if any moves occurred
       - Stop if no moves OR max_iterations reached
    3. Group nodes by final community_id
    4. Sort communities by smallest node_id, then sort nodes within each community
  - Modularity delta formula: Q = (edges_in / m) - (edges_total / m)^2
    - edges_in: edges within community after move
    - edges_total: total edges touching community after move
    - m: total edges in graph
  - Use ahash::AHashMap<i64, i64> for community assignments
  - Pre-calculate degree of each node (incoming + outgoing count)

  DO NOT implement multi-level aggregation (too complex for MVP).
  DO implement single-pass modularity optimization - simpler and effective.</action>

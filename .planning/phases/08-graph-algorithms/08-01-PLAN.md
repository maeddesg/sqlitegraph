  - Signature: pub fn betweenness_centrality(graph: &SqliteGraph) -> Result<Vec<(i64, f64)>, SqliteGraphError>
  - Implement Brandes' algorithm for unweighted graphs:
    1. For each node s in graph:
       - Run BFS from s to compute shortest paths
       - Track: number of shortest paths through each node
       - Accumulate: dependency of s on each node v
    2. Return centrality values sorted descending
  - Use AHashMap<i64, Vec<i64>> for predecessor lists during BFS
  - Use AHashMap<i64, usize> for path counts
  - Use AHashMap<i64, f64> for centrality accumulation
  - BFS uses graph.fetch_outgoing() for neighbor discovery
  - Handle disconnected components gracefully (some pairs have no path)

  DO NOT implement sampling approximation - use exact algorithm for simplicity.
  DO NOT optimize for large graphs yet - this is MVP implementation.</action>

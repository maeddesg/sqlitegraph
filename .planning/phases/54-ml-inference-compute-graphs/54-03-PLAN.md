# Research Summary (from 54-RESEARCH.md)
- Exact isomorphism: petgraph::algo::isomorphism::is_isomorphic_matching() - VF2-based boolean check
- Maximum Common Subgraph (MCS): NP-hard - use bounded enumeration (max_matches, timeout) for approximation
- Isomorphic regression testing (UIUC 2016): uses isomorphism + MCS for test prioritization
- Similarity score 0.0-1.0: 1.0 = isomorphic, 0.8-0.9 = highly similar, <0.5 = structurally different
- Graph edit distance (GED): simplified as 1.0 - mcs_similarity (no need for full GED computation)

# Key Technical Decisions
- Use petgraph::algo::isomorphism::is_isomorphic_matching() for exact isomorphism check
- MCS approximation: use smaller graph as pattern, find subgraph isomorphisms, return max size
- SimilarityResult contains:
  - isomorphic: bool (exact isomorphism check result)
  - mcs_similarity: f64 (0.0 to 1.0, max_common_nodes / max_total_nodes)
  - ged_distance: f64 (simplified: 1.0 - mcs_similarity)
  - mcs_size: usize (actual size of maximum common subgraph)
  - computation_time_ms: u128
- SimilarityBounds: max_matches for MCS enumeration (default 100), timeout_ms (default 5000)
- For program analysis: typically only need to detect "similar enough" (threshold 0.8-0.9)

# Anti-Patterns to Avoid
- Don't implement exact MCS - it's NP-hard, use bounded approximation
- Don't use Weisfeiler-Leman for exact isomorphism - it's incomplete (false positives)
- Don't compute full GED - simplified distance (1 - similarity) is sufficient

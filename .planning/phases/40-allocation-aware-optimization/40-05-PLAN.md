# 40-05: Delta-Aware Read Paths (Architecture Revision)

<objective>
Implement snapshot-aware read operations that check DeltaIndex for recent committed changes before falling back to mmap reads.

**Purpose:** The original plan 40-05 assumed WAL-based read overlay, but we're using delta-index architecture. This plan implements snapshot filtering via O(1) delta lookups, preserving the fast mmap path for checkpointed data.

**Output:** Read operations that overlay delta records on mmap base data for correct snapshot isolation.
</objective>

    ```rust
    impl GraphBackend for NativeGraphBackend {
        fn has_node(
            &self,
            graph_id: GraphId,
            node_id: NodeId,
            snapshot_id: SnapshotId,
        ) -> Result<bool, NativeError> {
            // Step 1: Check delta index for recent committed changes
            if let Some(delta) = self.wal_manager()
                .delta_index()
                .get_node_delta(node_id, snapshot_id)
            {
                // Apply delta record
                match &delta.record {
                    WalRecord::AddNode { .. } => return Ok(true),
                    WalRecord::DeleteNode { .. } => return Ok(false),
                    _ => {} // Other record types, fall through to mmap
                }
            }

            // Step 2: No visible delta, check mmap (checkpointed base)
            let graph_file = self.get_graph_file(graph_id)?;
            graph_file.node_exists(node_id) // Existing mmap-based read
        }

        fn get_node(
            &self,
            graph_id: GraphId,
            node_id: NodeId,
            snapshot_id: SnapshotId,
        ) -> Result<NodeRecord, NativeError> {
            // Step 1: Check delta index for recent committed changes
            if let Some(delta) = self.wal_manager()
                .delta_index()
                .get_node_delta(node_id, snapshot_id)
            {
                // Apply delta record
                match &delta.record {
                    WalRecord::AddNode {
                        node_id: nid,
                        labels,
                        properties,
                        ..
                    } if *nid == node_id => {
                        return Ok(NodeRecord {
                            id: node_id,
                            labels: labels.clone(),
                            properties: properties.clone(),
                        });
                    }
                    WalRecord::DeleteNode { .. } => {
                        return Err(NativeError::NodeNotFound(node_id));
                    }
                    _ => {} // Other record types, fall through to mmap
                }
            }

            // Step 2: No visible delta, read from mmap (checkpointed base)
            let graph_file = self.get_graph_file(graph_id)?;
            graph_file.read_node(node_id) // Existing mmap-based read
        }
    }
    ```

    **Key implementation notes:**
    - Check delta FIRST (O(1) HashMap lookup)
    - Apply delta if commit_lsn <= snapshot_id (visibility check already done in get_node_delta)
    - Fall back to mmap if no delta or delta not applicable
    - Preserve existing mmap read path (fast, unchanged)
    - AddNode delta -> node exists, return it
    - DeleteNode delta -> node doesn't exist, return error

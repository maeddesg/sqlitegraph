   ```rust
   #[derive(Debug, Clone, Copy)]
   pub struct PackedEdgeHeader {
       bits: u64,  // Packed representation
   }
   impl PackedEdgeHeader {
       // Pack: delta (32 bits) + type_offset (16 bits) + data_len (12 bits) + flags (4 bits)
       pub fn pack(delta: u32, type_offset: u16, data_len: u16, flags: u8) -> Self
       pub fn unpack_delta(&self) -> u32
       pub fn unpack_type_offset(&self) -> u16
       pub fn unpack_data_len(&self) -> u16
       pub fn unpack_flags(&self) -> u8
   }
   ```

2. Define flag bits:
   - bit 0: has_data (data payload present)
   - bit 1: large_delta (delta > u16, use extended encoding)
   - bit 2: null_data (data is JSON null)
   - bit 3: reserved

3. Add small data optimization:
   - If data <= 8 bytes, inline it in header extension
   - Otherwise use variable-length encoding

4. Modify serialization to use packed format:
   - Replace separate fields with packed header
   - Reduce per-edge overhead from ~24 bytes to ~12 bytes

AVOID: Do not apply bit-packing if graph has > 65535 edge types (type_offset exceeds u16).

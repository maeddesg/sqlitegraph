# Roadmap: SQLiteGraph

## Overview

SQLiteGraph v1.4 Code Quality & Features milestone resolves technical debt, completes missing features, fixes known bugs, and improves code quality. The roadmap delivers 4 phases: code quality foundation (clippy warnings, test safety), file structure refactoring (300 LOC guideline), snapshot features (LSN tracking, WAL integration), and bug fixes (HNSW pruning, gitignore enforcement).
